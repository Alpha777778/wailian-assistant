# 外链助手 Chrome 扩展

从 SEMrush 镜像站批量提取竞品外链，自动过滤博客评论机会，支持竞品发现与批量导出。

---

## 功能

### 提取
- 打开 SEMrush 反向链接页面后一键提取，自动翻页
- 按 Authority Score (AS) 过滤，整页低于阈值自动停止翻页
- 自动识别博客文章页、正文链接，综合评分排序
- 过滤社交媒体、搜索页、已丢失链接
- 提取结果导出为 Excel（UTF-8 BOM，Excel 直接打开不乱码）

### 已保存
- 按竞品域名分组展示，可折叠
- 导出全部 / 仅导出待提交 / 清空

### 提交
- 从已保存列表批量提交博客评论（调用 background.js 实现）
- 支持自定义评论内容或随机模板
- 可设置间隔时间、每批数量、最小 AS 分

### 竞品发现
1. 输入关键词 → 自动 Google 搜索 → 提取排名前 10 的域名
2. 手动编辑域名列表（每行一个）
3. 打开 SEMrush 镜像站并登录后，点击「开始批量导出」
4. 扩展自动在已打开的镜像站 tab 内逐个导航 → 点击 Follow 过滤 → 导出 Excel
5. 文件自动下载到系统下载目录

### 配置
- 评论者姓名、邮箱、网站 URL
- 最小 AS 分、提交间隔、每批数量
- AI 半自动提交支持三种模式：
  - 本机 Codex：复用 `~/.codex` 当前登录态
  - 本机 Claude：复用 `~/.claude` 当前登录态和 Base URL
  - 自定义 OpenAI 兼容 API

## 安装

1. 打开 Chrome，地址栏输入 `chrome://extensions/`
2. 右上角开启「开发者模式」
3. 点击「加载已解压的扩展程序」，选择本目录
4. 工具栏出现扩展图标即安装成功

---

## 使用流程

### 单域名提取
1. 在 SEMrush 镜像站（`sem.3ue.co`）打开某竞品的反向链接页面
2. 点击扩展图标 → 「提取」标签 → 「提取当前页外链」
3. 提取完成后在「已保存」查看结果，或直接「导出 Excel」

### 竞品批量发现
1. 「竞品发现」标签 → 输入关键词 → 「Google 搜索」（自动填入前 10 域名）
2. 手动打开 `sem.3ue.co` 并登录
3. 点击「开始批量导出」，等待自动完成
4. 下载目录会出现每个竞品对应的 Excel 文件

### 半自动提交
1. 先运行仓库里的 `start-local-ai-bridge.cmd`
2. 扩展「配置」页选择 `本机 Codex`、`本机 Claude` 或 `自定义 OpenAI 兼容 API`
3. 本机模式默认桥接地址使用 `http://127.0.0.1:8765`
4. 在「交叉分析」页导入 CSV / TXT / XLSX / XLS，或直接使用下方交叉分析结果
5. 点击「开始半自动」，插件会自动识别入口、自动填表，你负责登录、验证码和最终提交

## 文件结构

```
├── manifest.json          # 扩展配置（MV3）
├── popup.html             # 弹窗 UI
├── popup.js               # 弹窗逻辑
├── background.js          # Service Worker（评论提交）
├── local-ai-bridge.mjs    # 本机 AI 桥接服务
├── start-local-ai-bridge.cmd # 启动本机 AI 桥接
├── content_semrush.js     # 注入 SEMrush 页面的内容脚本
└── icon*.png              # 图标
```

---

## 注意事项

- 需要在 SEMrush 镜像站已登录的状态下使用
- 批量导出时请勿关闭镜像站 tab
- 关闭 popup 不会丢失日志和域名列表（已持久化到 storage）
- 本机 AI 模式需要保持 `start-local-ai-bridge.cmd` 对应的窗口持续运行
- 提交评论功能建议配合独立的 Python 提交工具使用，成功率更高
